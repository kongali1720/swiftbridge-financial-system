# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a potential security vulnerability in this project, please do NOT open a public issue.

Instead, please send an email to **security@younext.cloud** with the following details:

-   A description of the vulnerability.
-   Steps to reproduce the vulnerability (if possible).
-   Any potential impact you are aware of.

We will acknowledge receipt of your report within 48 hours and will provide a detailed response within 7 days. We will work with you to understand and resolve the issue.
